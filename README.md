# 最短のクリエイティブ・パイプライン

> 1枚の写真から、手に取れる感動を。

2D画像からAIで高品質な3Dモデルを生成し、3Dプリント可能なデータとして即座に書き出す「最短のクリエイティブ・パイプライン」です。

## 🌟 プロジェクトの概要

このプロジェクトは、**「3Dモデリングの知識がなくても、写真1枚から物理的なモノを作れる」**体験を提供するために開発されました。
既存の3D生成AIをコアエンジンに採用し、3Dプリントに必要な「厚み付け」「メッシュ補完」「ファイル変換」を自動化することで、誰でも簡単に「思い出の具現化」ができるサービスを目指しています。

### 既存サービス（Hitem3D等）との違い

- **スピード**: 数週間の制作期間を数十秒に短縮。
- **アクセシビリティ**: 高価な外注費用をかけず、AIによって無料〜低コストで実現。
- **一気通貫**: 3D化して終わりではなく、そのままプリント用形式（STL/OBJ）で書き出し、プレビューまで完結。

## ✨ 主な機能

- **AI 3D Generation**: 1枚の画像から、見えない裏側までAI（TripoSR / InstantMesh等）が推論して3D化。
- **Real-time Preview**: 生成されたモデルをブラウザ上で3D表示し、サイズ感を確認。
- **Print-Ready Export**: 3Dプリンタでそのまま扱える `.stl` / `.obj` 形式への自動変換。
- **AR Preview (Coming Soon)**: スマホをかざして、現実世界でのサイズ感をシミュレーション。
